## Building

The docs use [mdBook](https://github.com/rust-lang/mdBook) and [mdbook-last-changed](https://github.com/badboy/mdbook-last-changed). Install both tools with:

```shell
cargo install mdbook mdbook-last-changed
```

Then run

```shell
mdbook serve -o
```
